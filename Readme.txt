ЗАДАНИЕ: Доделать задание в группах: Создать калькулятор для работы с рациональными и комплексными числами, организовать
меню добавив в неё систему логирования.

Структура решения:
Модуль 1: controller - Модуль для связи и управления
Модуль 2: logger - Модуль логирования. Для записи в журнал полученных результатов
Модуль 3: operation_mode - Модуль для работы с комплексными и рациональными числами
Модуль 4: user_interface - Модуль для взаимодейстивя с пользователем и отображения результата
Модуль 5: main - Модуль для связи всех модулей

Как работать!
При запросе на тип чисел:
1 - для работы с комплексными числами
Пример ввода: 8+9j + 7-8j -> т.е операнды указываем без пробелов, но от оператора отделяем одним пробелом!
2 - для работы с остальными типами чисел
Пример ввода: 8 * 8, 7 + 5, 9 / 3 итд
